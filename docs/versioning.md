# Versioning

O Agenda Falante usa versionamento semântico simples na fase inicial.

## Convenção inicial

- `0.x` representa fase de desenvolvimento.
- `alpha` indica que a arquitetura ainda está em evolução.
- `1.x` fica reservado para a primeira implementação estável.

## Objetivo

Essa convenção deixa claro que a base do projeto ainda está sendo consolidada e que mudanças estruturais podem acontecer antes da estabilidade inicial.

## Arquivo `VERSION`

O arquivo `VERSION` contém a versão textual atual do projeto e serve como referência simples para a linha evolutiva do repositório.
